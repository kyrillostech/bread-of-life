**Bread of Life** is an open-source, authentic Coptic Orthodox Bible planner and reader web application. It is designed to help users read the entire Bible (Old and New Testaments) in a structured and spiritual way, either over one year or two years.

## ğŸŒŸ Features

*   **Custom Reading Plans:** Choose between a 1-year or 2-year reading journey.
*   **Daily Readings:** Automatically calculates daily readings based on your start date.
*   **Mixed Reading Schedule:** Balanced daily diet of Old Testament (Morning/Noon) and New Testament (Evening).
*   **Authentic Design:** Beautiful, dark-themed "Glass Monastic" interface inspired by Coptic heritage.
*   **Daily Wisdom:** Displays a daily quote from the Church Fathers or Scripture.
*   **Gamification:** Streak counter to encourage daily consistency.
*   **Offline Support:** Fully functional offline after initial load (PWA-ready architecture).
*   **Coptic Calendar:** Displays the current Coptic date alongside the Gregorian date.
